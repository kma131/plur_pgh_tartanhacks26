>> ABOUT PLUR PGH <<<

>> PROJECT OVERVIEW <<<

PLUR PGH is an interactive web application designed to allow users to discover their best suited nightlife venue in Pittsburgh. Built with a retro electronic aesthetic, this venue finder combines intelligent matching algorithms with a unique cyberpunk-inspired interface.

The application transforms a command-line venue finder into a modern web experience, featuring venue thumbnails, preference-based scoring, and a responsive design that works across all devices.

>> KEY FEATURES <<<

>> SMART MATCHING <<<

Advanced scoring algorithm that prioritizes user preferences, venue types, location proximity, and budget compatibility.

>> VISUAL DESIGN <<<

Retro electronic theme with lime green/black color scheme, pixel fonts, and cyberpunk-inspired styling.

>> RESPONSIVE LAYOUT <<<

Side-by-side venue cards with thumbnails positioned outside information boxes for optimal readability.

>> COMPREHENSIVE DATA <<<

Database of 200+ Pittsburgh venues with detailed information, pricing, types, and visual thumbnails.

>> COMMUNITY POSTS <<<

Share venue reviews, tips, and recommendations with the community. Discuss favorite spots and get insights from other users.

>> USER DASHBOARD <<<

Track your search history and past venue recommendations. Revisit previous results and compare your matched venues over time.
